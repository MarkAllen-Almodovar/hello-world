# Hello World Examples

This workspace contains simple "Hello, world!" programs in several programming languages.

Files:
- `hello.py` - Python
- `hello.js` - JavaScript
- `hello.php` - PHP
- `hello.c` - C
- `hello.java` - Java
- `hello.go` - Go
- `hello.rb` - Ruby
- `hello.rs` - Rust
- `hello_author.py` - Author example in Python

Run the programs using the appropriate language runtime or compiler.
